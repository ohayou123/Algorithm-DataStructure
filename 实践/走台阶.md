
>每次只能走1步或者2步，从低走到最高的10层，一共有多少种走法?

1. 到第10层，那么有9层的走法到10层+8层的走法到10层

2. 9层 = 8层 + 7 层..递归 f(n) = f(n-1)+f(n-1) 类似斐波那契。
3. 记忆化搜索把已有的保留防止重算



```
//n为几层台阶
if( n ==0||n==1||n==2){return n;}

int[] men = new int[];
men[0]=1;men[1]=2;

for(int i = 2;i<n;i++){
    men[i] = men[i-1]+men[i-2];
}
return men[n-1]
```
